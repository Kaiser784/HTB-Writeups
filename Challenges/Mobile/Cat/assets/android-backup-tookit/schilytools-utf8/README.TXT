Schily tools, but everything in UTF-8 instead of ISO-8859-1
(A collection of tools written or managed by Jörg Schilling)
Fixes characters not displayed properly after compilation, like ö
Original author's documentation: http://schilytools.sourceforge.net/

Use script "convert-iso88591-to-utf8.sh" to convert original files,
files in this project are already in UTF-8
